# Rock-Paper-Scissors-Lizard-Spock

## About the Project
This repository contains the source code for a **Rock-Paper-Scissors-Lizard-Spock** game. 

Our group developed this project during our early semesters as Computer Science students, as part of an IEEE Student Branch initiative. The primary goal of this project was to consolidate basic programming logic, conditionals, and software fundamentals right at the beginning of our degrees.

> **Note:** The terminal outputs and source code variables are written in Portuguese, as it was originally built for a local university project, but the game logic and functionality works!

## The Rules
- Scissors cuts Paper
- Paper covers Rock
- Rock crushes Lizard
- Lizard poisons Spock
- Spock smashes Scissors
- Scissors decapitates Lizard
- Lizard eats Paper
- Paper disproves Spock
- Spock vaporizes Rock
- (and as it always has) Rock crushes Scissors
